## 支出类别

| 类型 | 说明 | 审批要求 |
|:---|:---|:---|
| cTCC 兑换背书 | 锁定 TCC 为微小贡献者兑换 cTCC | lemondy 确认周汇总 PR 后执行 |
| 应急激励 | 突发性高价值贡献（如安全漏洞修复） | lemondy 裁量 + 广场公开理由 |
| 战略加码 | 对生态有结构性影响的贡献，加码分配 | lemondy 裁量 + 广场公开理由 |
| 社区运营 | AI 协作者周报编制、广场维护、新人引导 | 固定额度（按周结算） |
| 开源收购 | 购买闭源指令并开源 | 按 4.4 节流程（提案 → 议价 → 执行 → 记录） |
| 端点运营 | 服务器、域名、带宽等基础设施开支 | lemondy 裁量 |
